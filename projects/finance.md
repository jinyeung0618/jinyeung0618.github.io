## 시작 동기

주식을 처음 시작하면서 *매일 아침 무엇을 봐야 할지* 막막했다. 뉴스를 일일이 찾기엔 시간이 없고, 유료 리포트는 부담스럽다. 동시에 *과거 가격 흐름과 그 시점의 사건*을 시각적으로 연결해 보고 싶었다. 그래서 직접 만들었다.

핵심 원칙:

- **사실 기반** — 검색 결과·공개 데이터만. 추측·예측 단언 금지
- **간결함** — 출근길 5분 통독 가능한 길이
- **자동화** — 매일 사람 손 거치지 않고 발행
- **한 화면 = 한 섹션** — 두 섹션이 동시 보이면 정보 과부하. 탭으로 분리

## 구조

상단 탭으로 분리된 두 섹션. URL 해시(`#reports`, `#stats`)로 직접 진입·공유 가능.

### 📰 모닝 리포트
매일 평일 10:03 KST 자동 발행. *오늘 한 줄 → 큰 흐름 3가지 → Watchlist 6종목 → 오늘 알아야 할 것* 형식. 5분 통독.

### 📊 지수·종목 통계
종목별 상장일~현재 가격 흐름을 로그 스케일 라인 차트로. **방향이 바뀌는 변곡점(고점·저점)에만** 마커. 그 외 라인 위는 호버 무반응. 마커에 호버·터치하면 그 시점의 사건이 헤드라인 툴팁으로 뜸.

서브 탭 두 단계:
- 지역 (해외 / 국내) — `.tabset` segmented control
- 종목 (AAPL · TSLA …) — `.chip` 리스트, 다중 확장 가능

## 시스템 구성 (모닝 리포트)

```
              ↓

```

## 차트 구현 디테일

- **Chart.js** (CDN, ~20KB) — 정적 사이트에 적합. 빌드 스텝 없음
- **로그 스케일 Y축** — 41년간 약 4,000배 가격 차이를 한 화면에 시각화
- **Y축 눈금** — 주요 레벨만 (0.1 / 0.5 / 1 / 5 / 10 / 50 / 100 / **200** / 500 / 1000). 0.2, 0.3 같은 잔잔한 눈금 제거
- **X축 눈금** — 균등 간격 7개 + 마지막은 항상 현재 연도. 라벨 겹치면 직전 tick 자동 스킵
- **변곡점 마커** — 이벤트 dates에만 점 표시. `pointHitRadius` 20px 적용해서 모바일에서도 잘 잡힘
- **툴팁** — 날짜·가격 + (변곡점이면) `📌 헤드라인`. 변곡점 외에는 호버해도 무반응
- **데이터** — 참조점 → UTC 기준 월별 로그 보간 (~500 포인트). 실데이터 연동은 다음 단계
- **다크 모드 반응** — `MutationObserver`로 `data-theme` 변경 감지 → 차트 색상 자동 재렌더
- **탭 전환 시 재렌더** — `display:none` 상태였다가 표시될 때 canvas 사이즈 확보 후 재렌더

## 결정 로그

- **출처 URL 본문에 박지 않음** — 가독성 우선. 요청 시 별도 제공
- **수치 디테일 최소화** — 매출·EPS·capex 같은 분석가용 숫자 제외
- **추측·예측 단언 금지** — "~할 가능성이 높다" 같은 표현 금지. AI가 가장 빠지기 쉬운 함정
- **평일만 발행** — 토·일 휴장. 월요일에 주말 누적 뉴스 정리
- **6종목 watchlist** — 너무 많으면 리포트 길어지고 결정 피곤. 6개가 적정선
- **변곡점만 차트 마커** — 모든 사건이 아니라 *방향이 바뀐 순간*만. 신호 vs 노이즈 분리
- **변곡점 외 호버 무반응** — 클러터 방지. 라인 위 어디서나 툴팁 뜨면 정보 산만
- **한 화면 = 한 섹션** — 탭으로 분리. URL 해시 동기화로 공유 가능
- **탭 컴포넌트 통일** — `.tabset` (그룹 segmented, section/region) + `.chip` (독립 pill, ticker). 두 패턴으로 hierarchy 표현
